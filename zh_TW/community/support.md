# 尋求支持

如果你閱讀這套文檔時，仍然對產品使用存在疑問和建議，可嘗試以下方式尋求支持。我們的團隊與社區會竭盡所能的為你提供幫助。

### 社區支持

{% hint style="info" %}
請不要將涉及你的 Dify 賬戶信息與其它密鑰信息發至社區，我們的支持人員也不會索要您的賬戶信息。
{% endhint %}

* 在 [Github](https://github.com/langgenius/dify) 上提交 Issue
* 加入 [Discord ](https://discord.gg/8Tpq4AcN9c)社群
